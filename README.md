 読書好きの人のためのWebアプリ<br >
 呼んだ本を記録する<br >
   ＋遊び心でキャラクター要素（呼んだ本のジャンルに応じてキャラクターの見た目が変化します）

# 機能一覧
- ユーザー登録、ログイン機能
- 検索機能
  - Google Books API
- 記録機能
  - 索引機能
  - 昇順・降順機能
- エフェクト
  - スクロール
  - 浮き出し
  - 画像切り替え
  - ポップアップ
